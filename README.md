## Description

This project is a pixel-perfect recreation of Dipper Pines, the young adventurer from the animated series Gravity Falls, created entirely using HTML and CSS. No images are used; everything is drawn with divs, borders, box-shadows, and CSS shapes.

## Features

- Pure CSS art
- Responsive design (uses vmin units)
- Accessible (includes aria labels)
- Lightweight (no external dependencies except Google Fonts for the name)

## How it Works

The portrait is composed of multiple `<div>` elements, each styled with CSS to form parts of the face, hair, hat, and clothing. Techniques include:

- CSS variables for colors
- Box-shadows for shading
- Clip-path for complex shapes
- Gradients and borders for details

## Usage

Simply clone the repository and open `index.html` in a browser, or host it on a web server.

```bash
git clone https://github.com/BinaryVortex/Dipper-Pines-CSS.git
cd Dipper-Pines-CSS
open index.html
```
